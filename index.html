Persönlicher Kalender-Abonnement-Link für jeden Mitarbeiter

📲 Anleitung: Kalender abonnieren

1
Suche unten deinen Login und klicke auf den passenden Button für dein Gerät.
2
iPhone / iPad: Klick auf „iOS" → Kalender-App öffnet sich → „Abonnieren" tippen → fertig.
Oder manuell: Einstellungen → Kalender → Accounts → Account hinzufügen → Andere → Kalenderabo hinzufügen → Link einfügen
3
Google Kalender: Klick auf „Google" → Google Kalender öffnet sich → „Hinzufügen" klicken.
Oder manuell: calendar.google.com → Andere Kalender (+) → Per URL → Link einfügen → Kalender hinzufügen
4
Outlook Desktop: Datei → Kontoeinstellungen → Internetkalender → Neu → Link einfügen (📋 Kopieren-Button)
5
Updates: Der Kalender aktualisiert sich automatisch (iOS: täglich, Google: alle 12h). Bei Planänderungen einfach abwarten – du musst nichts tun.
