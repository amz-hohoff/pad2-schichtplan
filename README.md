# PAD2 RME Schichtplan 2026

Kalender-Abonnements für das PAD2 RME Team.

## Abonnieren

👉 **[Zur Übersicht aller Links](https://amz-hohoff.github.io/pad2-schichtplan/)**

Jeder Mitarbeiter findet dort seinen persönlichen Link für:
- 📱 iOS Kalender (direkter Abo-Link)
- 🌐 Google Kalender
- 🗓 Outlook Desktop

## Struktur

```
shift-a/   → Shift A (18 Mitarbeiter)
shift-b/   → Shift B (19 Mitarbeiter)
shift-c/   → Shift C (19 Mitarbeiter)
index.html → Übersicht mit allen Links
```

## Updates

Bei Planänderungen werden die ICS-Dateien aktualisiert.  
Der Kalender aktualisiert sich automatisch auf allen Geräten (iOS: ~24h, Google: ~12h).
